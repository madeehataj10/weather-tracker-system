# weather-tracker-system
The Weather Tracker System is an academic project designed to provide real‑time weather information based on user input or location. The project focuses on understanding API integration, data retrieval, and presenting weather details such as temperature, humidity, and weather conditions in a user‑friendly format. 

 Weather Tracker System
